table 태그

  • HTML 에서 표를 출력할 때 사용한다.
  • table, tr, th, td 태그로 구성된다.
  • table : 테이블 전체를 의미하는 태그
  • tr (table row) : 테이블 행 하나를 의미하는 태그
  • th (table header) : 테이블 제목 칸을 의미하는 태그
  • td (table data) : 테이블 내용 칸을 의미하는 태그
