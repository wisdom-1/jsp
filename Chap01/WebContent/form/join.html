회원 가입 페이지

Form 태그

  • 클라이언트의 입력을 수집하기 위해 사용하는 태그
  • 사용자가 입력한 내용을 서버로 전송할 수 있다.
  • action 속성 : 사용자의 입력이 전송될 URL을 설정한다.(설정이 없으면 현재페이지)
  • method 속성 : action에 설정된 URL로 데이터를 보낼 때의 HTTP method를 설정한다.

  • GET 방식 - 데이터를 URL 맨 뒤에 실어 보내는 방식 (default)
    보낼때, ?name=value1&name2=value2&.... 의 식
    GET방식은 길이 제한이 있어서 너무 긴 데이터를 보내면 짤릴 위험이 있다. (데이터 손실 위험)
    사용자가 해당 페이지를 즐겨찾기하기를 원한다면 GET방식을 사용하는 것이 좋다.

  • POST 방식 - 요청(request)의 body라는 부분에 데이터를 실어 보내는 방식 /주소에는 안보임
    POST 방식은 길이 제한이 없어서 글 내용 같은 긴 데이터를 보내는데에 적합하다.
